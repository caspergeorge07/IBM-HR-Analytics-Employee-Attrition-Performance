# TS Academy Capstone Project — Group 04

## IBM HR Analytics Employee Attrition

### Project Overview

This project investigates employee attrition patterns using the IBM HR Analytics Employee Attrition dataset.

The aim is to:
- Understand factors contributing to employee attrition
- Segment employees using clustering
- Build machine learning models to predict attrition
- Apply imbalance handling techniques
- Generate business recommendations for HR retention strategies
